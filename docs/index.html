Features

One Windows package

Users download a release ZIP and run the included executable without installing developer tools.

LDPlayer control

Open Game starts LDPlayer when needed and opens Clash of Clans through the existing ADB flow.

Live logs

Backend output streams into the GUI so connection and detection errors are visible.

Manual tools

Return Home, Screenshot, Detect Loot, Manual Attack, and Reload Config are available from the app.

Bases and armies

Save Clash base and army links locally as JSON and copy or open them from the GUI.

Release workflow

GitHub Actions can build and publish the Windows ZIP from a version tag.

Quick Start

Download CoC-Farm-Bot-Windows.zip from the latest GitHub Release.
Extract the ZIP and run CoC Farm Bot.exe.
Set LDPlayer to 1920 x 1080 and enable ADB debugging.
Click Open Game, then click Start.

FAQ

Do I need Python?

No. The Windows release ZIP is for normal users and includes the runtime.

Can I move only the exe?

No. Keep the exe together with the _internal folder from the ZIP.

What emulator?

This build targets LDPlayer using ADB and ldconsole.
