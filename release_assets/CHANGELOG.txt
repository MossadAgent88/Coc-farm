CoC Farm Bot Changelog
======================

v1.5.5 - Python 3.14 Windows EXE Release
----------------------------------------

- Built the Windows EXE with Python 3.14.
- Updated source launcher, local build scripts, and GitHub Actions to use Python 3.14.
- Added Python runtime logging and unsupported-version errors.
- Kept Python 3.15 marked experimental because PySide6 currently blocks it.

v1.0.0 - Windows EXE Release
----------------------------

- Added a Windows release package for normal users.
- Added the Claude Design GUI as the app frontend.
- Added live GUI log streaming from the backend bot process.
- Added Bases and Armies tabs with JSON persistence.
- Connected Manual actions for Open Game, Return Home, Screenshot, Detect Loot,
  Manual Attack, and Reload Config.
- Added LDPlayer startup/open-game flow through the existing ADB/ldconsole system.
- Added release documentation, quick-start instructions, and GitHub Actions
  release packaging.

Unofficial project. Not affiliated with Supercell.
